# GameOfLife-Simulator
A program that is used to simluate Game of life, one of the most famous classical 2-D games. The game is zero-player meaning that the evolution of cells are determined
by their neighbours. In this game, user are provided options for changing the color of cells and the rate evolution. The default rate of evolution is 1 generation/1000ms.
In addition, user can also pause/resume the game and re-create the game board with different dimension.
